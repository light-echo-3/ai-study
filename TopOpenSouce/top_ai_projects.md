# GitHub 热门 AI 开源项目精选

本列表结合了 GitHub 历史总星标数 (Stars) 最高的顶级 AI 项目，以及近期（2026年3月）爆发式增长的现象级 Trending 项目。

**当前行业趋势：** Agent（智能体）生态全面爆发，围绕 AI Agent 的工具链、沙箱、技能框架、基础设施正在全方位开花。

---

## 🔥 近期现象级 Trending 项目 (Agent 生态大爆发)

### 🌟 绝对王者与开发方法论
1. **[openclaw/openclaw](https://github.com/openclaw/openclaw)** — ⭐ 315k 
   * **描述**: 本月狂涨 12 万星的绝对王者。你的专属个人 AI 助手。支持任何操作系统和平台。
2. **[obra/superpowers](https://github.com/obra/superpowers)** — ⭐ 86.9k
   * **描述**: Agentic Skills 框架与软件开发方法论。定义了一套 AI Agent 协作开发的最佳实践。
3. **[badlogic/pi-mono](https://github.com/badlogic/pi-mono)** — ⭐ 22.5k
   * **描述**: AI Agent 工具包：包含 coding agent CLI + 统一 LLM API + TUI/Web UI。OpenClaw 生态里的 Pi agent 就是基于此构建。

### 🚀 Agent 框架与运行生态
4. **[bytedance/deer-flow](https://github.com/bytedance/deer-flow)** — ⭐ 29.5k
   * **描述**: 字节跳动开源的 SuperAgent 框架。具备研究、写代码、创作能力，内置沙箱、记忆系统、工具和子 Agent 调度。
5. **[volcengine/OpenViking](https://github.com/volcengine/OpenViking)** — ⭐ 5.9k
   * **描述**: 字节火山引擎推出的 Agent 上下文数据库。专门用于管理 Agent 的记忆、资源和技能，原生支持 OpenClaw。
6. **[huggingface/skills](https://github.com/huggingface/skills)** — ⭐ 9.1k
   * **描述**: HuggingFace 官方推出的 AI Skills（技能）框架。
7. **[alibaba/OpenSandbox](https://github.com/alibaba/OpenSandbox)** — ⭐ 7.5k
   * **描述**: 阿里开源的 AI 通用沙箱平台。专门为 Coding Agent 和 GUI Agent 提供安全的系统级执行环境。

### 🧠 前沿新概念与黑科技
8. **[ruvnet/RuView](https://github.com/ruvnet/RuView)** — ⭐ 37.2k
   * **描述**: 极其炸裂的黑科技：用纯 WiFi 信号（无需摄像头）进行人体姿态估计和生命体征监测。底层采用 Rust 编写。
9. **[moeru-ai/airi](https://github.com/moeru-ai/airi)** — ⭐ 32.8k
   * **描述**: 自托管 AI 伴侣，号称"灵魂容器"。支持实时语音聊天、打 Minecraft、打 Factorio。
10. **[666ghj/MiroFish](https://github.com/666ghj/MiroFish)** — ⭐ 28.2k
    * **描述**: 群体智能引擎，主打"预测万物"。这是一个非常有意思的通过群体 AI 智能来进行事件预测的新方向。
11. **[koala73/worldmonitor](https://github.com/koala73/worldmonitor)** — ⭐ 38.8k
    * **描述**: AI 驱动的全球实时情报仪表盘。用于地缘政治监控和基础设施追踪。

### 🛠️ 实用基础设施与工具
12. **[alibaba/zvec](https://github.com/alibaba/zvec)** — ⭐ 9k
    * **描述**: 阿里开源的轻量级向量数据库。采用 C++ 编写，主打极致的查询速度。
13. **[clockworklabs/SpacetimeDB](https://github.com/clockworklabs/SpacetimeDB)** — ⭐ 23.6k
    * **描述**: 数据库 + 服务器合一的新型架构。Rust 编写，口号是 "Development at the speed of light"。
14. **[p-e-w/heretic](https://github.com/p-e-w/heretic)** — ⭐ 11.3k
    * **描述**: 叛逆的实用工具："异端"。用于全自动解除各大 LLM 厂商的审查限制 (Jailbreak)。

### 📚 学习资源与最佳实践
15. **[shareAI-lab/learn-claude-code](https://github.com/shareAI-lab/learn-claude-code)** — ⭐ 25.3k
    * **描述**: 从零教你实现一个 Claude Code 级别的 AI Agent ("Bash is all you need")，是学习 Agent 原理的极佳教材。
16. **[shanraisshan/claude-code-best-practice](https://github.com/shanraisshan/claude-code-best-practice)** — ⭐ 17.4k
    * **描述**: Claude Code 最佳实践指南，口号是 "practice made claude perfect"。
